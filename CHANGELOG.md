# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - 2026-02-05

Initial release.

### Added

- `CitableImage` class identifying images by `Cite2Urn`.
- `IIIFService` class configuring a IIIF service that can recognize image citation by URN.
- functions to compose URLs and retrieve binary image data from a `IIIFService` using URN references
